# Human Resource Management System

A modern Human Resource Management System (HRMS) built with **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**.

## Features

* 👥 Employee Management
* 📅 Attendance Tracking
* 📝 Leave Management
* 💰 Salary Management
* 🌙 Light/Dark Theme Support
* 📱 Responsive User Interface
* ⚡ Fast performance with Next.js App Router

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* shadcn/ui
* Turbopack\
* Ag grid

## Getting Started

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.
